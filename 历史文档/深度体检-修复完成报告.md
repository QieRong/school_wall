### 1. ✅ 漂流瓶 - 冷却计时器前后端同步

**修复内容**:

- 实现了 `syncCooldownWithBackend()` 函数，优先从后端获取冷却状态
- 添加了降级机制，后端失败时使用本地状态
- 修改了 `onMounted` 逻辑，优先同步后端而非本地状态

**修改文件**: `vue-frontend/src/views/Bottle.vue`

**关键代码**:

```javascript
const syncCooldownWithBackend = async () => {
  if (!currentUser.value) return;

  try {
    const res = await request.get("/bottle/cooldown-status", {
      params: { userId: currentUser.value.id },
    });
    if (res.code === "200" && res.data && res.data.remainingSeconds > 0) {
      startCooldown(res.data.remainingSeconds);
    } else {
      localStorage.removeItem(COOLDOWN_KEY);
      cooldown.value = 0;
      cooldownEndTime.value = null;
    }
  } catch (e) {
    console.error("同步冷却状态失败，使用本地状态", e);
    restoreCooldown();
  }
};
```

**关键代码**:

```javascript
const checkReadRequirement = () => {
  const len = paragraphs.value.length;
  const requiredReads = [];
  for (let i = Math.max(0, len - 3); i < len; i++) {
    requiredReads.push(i);
  }

  const unreadIndexes = requiredReads.filter(
    (idx) => !readParagraphs.value.has(idx)
  );

  if (unreadIndexes.length > 0) {
    return {
      canContinue: false,
      message: `请先阅读最近 ${unreadIndexes.length} 段内容`,
      firstUnread: unreadIndexes[0],
    };
  }

  return { canContinue: true };
};
```
