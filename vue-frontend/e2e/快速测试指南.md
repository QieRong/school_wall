# Playwright E2E测试快速指南

## 一、测试前准备

### 1. 启动服务
```bash
# 终端1：启动后端（端口19090）
cd springboot
mvn spring-boot:run

# 终端2：启动前端（端口3000）
cd vue-frontend
npm run dev
```

### 2. 导入测试数据
```bash
# 在MySQL中执行
mysql -u root -p biaobaiqiang < vue-frontend/e2e/setup-test-data.sql
```

### 3. 测试账号
- 普通用户：`2212040241` / `123456`
- 管理员：`admin` / `123456`

## 二、运行测试

### 方式1：命令行运行（推荐）
```bash
cd vue-frontend

# 运行所有测试
npx playwright test

# 运行特定模块
npx playwright test 01-auth.spec.js
npx playwright test 02-post.spec.js

# 运行单个测试
npx playwright test 01-auth.spec.js:13
```

### 方式2：UI模式（可视化调试）
```bash
cd vue-frontend
npx playwright test --ui
```

在UI界面中：
1. 左侧选择要运行的测试
2. 点击绿色三角按钮开始运行
3. 查看右侧的执行过程和结果

### 方式3：调试模式
```bash
# 以调试模式运行特定测试
npx playwright test 01-auth.spec.js --debug
```

## 三、查看测试结果

### 查看HTML报告
```bash
npx playwright show-report
```

### 查看失败截图
失败的测试会自动保存截图和视频到：

### 查看Trace
```bash

```

## 五、常见问题

### Q1: 测试失败提示"TimeoutError"
**原因：** 页面加载慢或元素未找到
**解决：**
1. 检查前后端服务是否正常运行
2. 检查网络连接
3. 增加等待时间（已在代码中设置）

### Q2: 找不到按钮或元素
**原因：** DOM结构与选择器不匹配
**解决：**
1. 查看`按钮文本映射表.md`确认正确的文本
2. 使用Playwright Inspector调试：`npx playwright test --debug`

### Q3: 测试数据冲突
**原因：** 多次运行测试导致数据重复
**解决：**
1. 重新导入测试数据
2. 使用随机生成的账号（已在代码中实现）

### Q4: 管理员测试失败
**原因：** 管理员账号未正确配置
**解决：**
1. 确认数据库中admin账号存在
2. 确认角色权限正确设置

## 六、测试最佳实践

### 1. 按顺序运行
建议按文件编号顺序运行测试：
```bash
npx playwright test 01-auth.spec.js
npx playwright test 02-post.spec.js
# ... 依次类推
```

### 2. 单独调试失败用例
```bash
# 只运行失败的测试
npx playwright test --last-failed
```

### 3. 并行运行（谨慎使用）
```bash
# 使用2个worker并行运行
npx playwright test --workers=2
```

注意：并行运行可能导致数据冲突，建议先单线程验证。

### 4. 生成测试报告
```bash
# 运行测试并生成报告
npx playwright test --reporter=html

# 查看报告
npx playwright show-report
```

## 八、持续集成建议

### GitHub Actions配置示例
```yaml
name: E2E Tests
on: [push, pull_request]
jobs:
  test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - uses: actions/setup-node@v3
      - name: Install dependencies
        run: cd vue-frontend && npm ci
      - name: Install Playwright
        run: cd vue-frontend && npx playwright install --with-deps
      - name: Run tests
        run: cd vue-frontend && npx playwright test
      - uses: actions/upload-artifact@v3
        if: always()
        with:
          name: playwright-report
          path: vue-frontend/playwright-report/
```
