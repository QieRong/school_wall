
**确认后端已启动**：访问 http://localhost:19090，应该能看到响应

## 🔍 测试失败排查

### 问题1：所有测试都失败

**原因**：后端或前端服务未启动

**解决**：
1. 检查后端是否在运行（访问 http://localhost:19090）
2. 检查前端是否在运行（访问 http://localhost:3000）

---

### 问题2：登录测试失败

**原因**：测试用户不存在

**解决**：执行 `setup-test-data.sql` 脚本创建测试用户

---

### 问题3：测试超时

**原因**：网络慢或服务响应慢

**解决**：在 `playwright.config.js` 中增加超时时间：

```javascript
timeout: 60 * 1000, // 改为60秒
```

---

### 问题4：端口冲突

**原因**：3000端口被占用

**解决**：
1. 修改 `vite.config.js` 中的端口
2. 同时修改 `playwright.config.js` 中的 `baseURL`

## 💡 提示

1. **首次运行**：建议先运行单个测试文件，确认环境正常
2. **调试模式**：使用 `--debug` 参数可以逐步调试
3. **无头模式**：使用 `--headed` 参数可以看到浏览器界面
4. **并行执行**：默认会并行运行，可能导致数据冲突，建议先串行测试

```bash
# 串行运行（推荐首次测试）
npx playwright test --workers=1

# 显示浏览器界面
npx playwright test --headed

# 调试模式
npx playwright test --debug
```
