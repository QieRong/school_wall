## 🔧 修复策略

### 方案1：使用更宽松的选择器
```javascript
// 不推荐：精确匹配
await page.click('button:has-text("立 即 登 录")')

// 推荐：部分匹配（忽略空格）
await page.click('button:has-text("登录")')
```

### 方案2：使用aria-label
```javascript
// 最推荐：使用语义化属性
await page.click('button[aria-label="登录"]')
await page.click('button[aria-label="注册"]')
```

### 方案3：使用CSS类名
```javascript
// 备用方案：使用类名
await page.click('button.bg-primary:has-text("登录")')
```

## ⚠️ 注意事项

1. **空格问题**：中文按钮文本中可能有空格，需要精确匹配
2. **动画问题**：按钮可能有动画效果，需要等待稳定
3. **禁用状态**：表单验证可能导致按钮禁用，需要确保输入有效
4. **加载状态**：按钮可能显示"处理中..."，需要等待完成
